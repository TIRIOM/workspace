for 문 

    
        작성법 
        for (초기식 ; 조건식 ; 증감식){
            반복 수행할 코드 
        }

        
    
            
  • 초기식 : 반복문이 수행될 때 "처음에 딱 한번만 실행하는 구문 
    • 
            
  • 조건식 : "반복문이 수행할 조건"을 작성하는 구문 
                
      
                    
    • 조건식이 true 일 경우 해당 반복을 실행
      • 
                    
    • 조건식이 false가 되는 순간 반복문을 탈출
      • 
                
    
                
      
                    
    • 주의점 1 : const 쓰면 변수 수정 안되서 X, 대신 let 써야 함 
      • 
                    
    • 주의점 2 : I++를 쓰던지 , 식을 쓸거면 =를 활용해 넣어줘야 함. 아님 += (복합연산자) 쓰던지 
      • 
                
    
            
    • 
            
  • 증감식 : 반복문을 제어하는 변수 값을 증감시키는 구문
