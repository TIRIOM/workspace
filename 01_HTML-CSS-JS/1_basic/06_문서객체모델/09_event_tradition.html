이벤트(event)

        1. 이벤트(Event) : 동작, 행위 
        -> 브라우저에서의 동작 , 행위 : click, keyup , keydown, changew, submit ,,.....이런 거를 이벤트라고 한다. 
        
        2. 이벤트 리스터 (Event Listener)
        -> 이벤트가 발생하는 것을 대기하고 있다가, 
           이벤트가 발생하는 것이 감지되면, 연결된 기능(함수)를 수행하는 것 

           ex) onload, onclick, onkeyup .... (on + 이벤트명 형식)

           on이벤트명 = "함수명()"

        3. 이벤트 핸들러 (Event handler)
        -> 이벤트 리스터에 연결된 기능으로 
           이벤트 발생 시 수행하고자 하는 내용을 작성해둔 함수 (function)

[이벤트 모델 종류

1. 인라인 이벤트 모델

        요소 내부에 이벤트를 작성하는 방법 
        on이벤트명 = "함수명()"

2. 고전 이벤트 모델

        요소가 가지고 있는 이벤트 속성에 
        이벤트 핸들러를 연결하는 방법 
        -> script에만 이벤트 관련 코드를 작성할 수 있다.

3. 표준 이벤트 모델

        - W3C (웹 표준 재정 단체)에서 
        공식적으로 지정한 이벤트 모델

        - 한 요소에 여러 이벤트 핸들러 설정 가능 
        -> 고전 이벤트 모델 단점 보안
